FormGenie Documentation

A powerful form creation and customization tool for your web applications

Introduction

FormGenie is a flexible and easy-to-use JavaScript library that allows you to quickly generate and customize forms for your web applications. Whether you need simple input forms or complex forms with custom validation, FormGenie has you covered.

Installation

To get started with FormGenie, you can include it via CDN link and include the other libraries by following the step below

Step 1: Adding CDN

<script src="https://code.jquery.com/jquery-3.6.4.min.js"></script>
<script src="https://cdn.jsdelivr.net/gh/Abstechs/Abstech@main/formgenie.min.js"></script>

Step 2: Other libraries

FormGenie utilizes jQuery internally and tailwind CSS for styling by default, you can change it to bootstrap (optional). No need if your project already included any of the library. These are the links to includes in your projects:

<!-- Add these to the head tag -->:
<-- Tailwind CSS -->
<link href="https://cdn.jsdelivr.net/npm/tailwindcss@2.2.19/dist/tailwind.min.css" rel="stylesheet">
<--  Toast library -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/izitoast/dist/css/iziToast.min.css">

<!-- Add these before the closing body tag </body> -->
<script src="https://cdn.jsdelivr.net/npm/sweetalert2@11"></script>
    
<!-- iziToast JS -->
<script src="https://cdn.jsdelivr.net/npm/izitoast/dist/js/iziToast.min.js"></script>

Step 3: Local Installation

If you prefer to download FormGenie, simply download the library from FormGenie GitHub, then include it in your project by adding this:

<script src="path/to/your/formgenie.min.js" defer></script>

Features

  • Supports various input types (text, email, password, select, etc.)
  • Customizable form structure and layout
  • Built-in validation for common fields (email, required fields, etc.)
  • Customizable themes (Tailwind CSS, Bootstrap, etc.)
  • Dynamic form submission with AJAX support
  • Easy integration with existing projects
  • Support for custom form field classes and styles

Usage

FormGenie is very easy to use. Simply call the FormGenie.create() method, passing in the required configuration options.

Basic Usage Example

FormGenie.create({
    container: "#form-container",  // The container where the form will be rendered
    fields: [
        { name: "fullName", label: "Full Name", type: "text", required: true, placeholder: "Enter your full name" },
        { name: "email", label: "Email Address", type: "email", required: true, placeholder: "Enter your email" }
    ],
    submitText: "Submit",  // Text on the submit button
});

Examples

Example 1: Basic Form

FormGenie.create({
    container: "#form-container-1",
    fields: [
        { name: "fullName", label: "Full Name", type: "text", required: true, placeholder: "Enter your full name" },
        { name: "email", label: "Email Address", type: "email", required: true, placeholder: "Enter your email" }
    ],
    submitText: "Submit"
});

Example 2: Form with Custom Validation

FormGenie.create({
    container: "#form-container-2",
    fields: [
        {
            name: "email",
            label: "Email Address",
            type: "email",
            required: true,
            placeholder: "Enter your email",
            validate: function(value) {
                return /^[\w-]+(\.[\w-]+)*@([\w-]+\.)+[a-zA-Z]{2,7}$/.test(value);  // Custom email validation
            }
        }
    ],
    submitText: "Submit"
});

Example 3: Form with Select and Radio Fields

FormGenie.create({
    container: "#form-container-3",
    fields: [
        { name: "fullName", label: "Full Name", type: "text", required: true, placeholder: "Enter your full name" },
        {
            name: "gender",
            label: "Gender",
            type: "select",
            required: true,
            options: [
                { value: "male", label: "Male" },
                { value: "female", label: "Female" },
                { value: "other", label: "Other" }
            ]
        },
        {
            name: "preferredContact",
            label: "Preferred Contact Method",
            type: "radio",
            required: true,
            options: [
                { value: "email", label: "Email" },
                { value: "phone", label: "Phone" }
            ]
        }
    ],
    submitText: "Submit"
});

Example 4: Form with Custom Themes

FormGenie.create({
    container: "#form-container-4",
    fields: [
        { name: "fullName", label: "Full Name", type: "text", required: true, placeholder: "Enter your full name" }
    ],
    theme: "tailwind",  // Apply Tailwind CSS theme
    submitText: "Submit"
});

Example 5: Form with AJAX Submission

FormGenie.create({
    container: "#form-container-5",
    fields: [
        { name: "fullName", label: "Full Name", type: "text", required: true, placeholder: "Enter your full name" },
        { name: "email", label: "Email Address", type: "email", required: true, placeholder: "Enter your email" }
            ],
    submitText: "Submit",
    onSubmit: function(data) {
        $.ajax({
            url: "/submitForm",
            method: "POST",
            data: data,
            success: function(response) {
                alert("Form submitted successfully!");
            },
            error: function(error) {
                alert("Error submitting form.");
            }
        });
    }
});
