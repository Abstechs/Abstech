Overview

FormGenie is a lightweight, AI-powered form generation library created by AbsTech. It simplifies dynamic and customizable form creation, offering flexibility, responsiveness, and beauty in one package. Tailored for developers who value efficiency and elegance, FormGenie is perfect for modern web applications.

Installation

You can integrate FormGenie easily into your project with the following steps:

<script src="formgenie.min.js"></script>

Ensure that jQuery is included in your project. FormGenie automatically fetches it if not available.

Usage

Initialize FormGenie and create forms dynamically using the following code: 

FormGenie.create({
  container: "#form-container",
  fields: [
    { name: "name", type: "text", placeholder: "Full Name", required: true },
    { name: "email", type: "email", placeholder: "Email Address", required: true },
    { name: "password", type: "password", placeholder: "Password", required: true },
    { name: "confirmPassword", type: "password", placeholder: "Confirm Password", required: true }
  ],
  theme: "tailwind",
  submitText: "Register"
});

The container should be a valid selector for your form's parent element.

Features

  • ✔ AI-powered form validation for email, phone, and password matching.
  • ✔ Fully customizable fields with support for custom attributes.
  • ✔ Multiple field types, including checkboxes, radios, and selects.
  • ✔ Default styling using Tailwind CSS or Bootstrap.
  • ✔ Responsive design for all screen sizes.
  • ✔ Animated field labels and inputs for a modern, intuitive experience.
  • ✔ Option to add 'genie' prefixed animations for additional customization.
